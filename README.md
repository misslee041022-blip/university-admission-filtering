# University Admission Filtering System

(대입 최저학력기준 자동 필터링 시스템)

## Project Description

This program filters universities based on the user's CSAT(Suneung) scores and minimum academic standards.
It utilizes Python `pandas` for data analysis and `tkinter` for the GUI.

## Tech Stack

- Python 3.9+
- Pandas (Data Processing)
- Openpyxl (Excel Support)
- Tkinter (GUI)

## How to Run

1. Install dependencies: `pip install pandas openpyxl`
2. Run the script: `python main.py`
3. Load the `my_database.xlsx` file and input your scores.
